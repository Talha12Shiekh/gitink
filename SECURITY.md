# Security Policy

## Supported Versions

We actively monitor and patch security vulnerabilities in the following versions of GitInk:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

---

## Reporting a Vulnerability

If you discover a security vulnerability in GitInk, please **do not open a public GitHub issue**. Instead, report it privately to our security team.

### How to Report
Please email your report to **tk.shiekh4567@gmail.com**.

In your report, please include:
1. A detailed description of the vulnerability.
2. Steps or proof-of-concept (PoC) code to reproduce the issue.
3. The potential impact of the vulnerability.

We will review your submission and respond within 48 hours to coordinate a security patch.
